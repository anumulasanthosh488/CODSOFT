# 💳 Credit Card Fraud Detection Dashboard


An interactive **Machine Learning-based Credit Card Fraud Detection Dashboard** developed using **Python**, **Streamlit**, and the **Random Forest Classifier**. This application helps identify fraudulent credit card transactions through real-time prediction and interactive visual analytics.

# 📖 Project Overview

Credit card fraud is one of the biggest challenges in digital banking. This project applies Machine Learning techniques to classify transactions as **Fraud** or **Normal**.

The application provides an interactive dashboard that allows users to:

- Upload transaction datasets
- Predict fraudulent transactions
- Visualize fraud statistics
- Analyze model performance
- Download prediction results

# ⚙ Installation

