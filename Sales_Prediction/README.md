# 📈 Sales Prediction Dashboard

An AI-powered Sales Prediction Dashboard built using **Python, Streamlit, and Machine Learning**. This project predicts product sales based on TV advertising budgets using a Simple Linear Regression model and provides an interactive dashboard for data analysis and visualization.

---

## 🚀 Features

- 📂 Upload Advertising Dataset (CSV)
- 📊 Interactive Dashboard
- 📈 Sales Analytics using Plotly
- 🤖 Machine Learning Prediction
- 📉 Model Performance Metrics
- 📋 Dataset Preview
- 📊 Correlation Matrix
- 📈 Feature Analysis
- 🎯 Sales Prediction Gauge
- 📄 Download Prediction Report
- 🌙 Professional Dark Theme UI

---

## 🚀 How to Run

### 1. Clone the Repository

```bash
git clone https://github.com/yourusername/Sales_Prediction_Project.git
```

### 2. Open Project Folder

```bash
cd Sales_Prediction_Project
```

### 3. Install Dependencies

```bash
pip install -r requirements.txt
```

### 4. Run the Application

```bash
streamlit run app.py
```

### 5. Open Browser

```
http://localhost:8501
```

Upload `advertising.csv` and explore the dashboard.
